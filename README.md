# 💎 M_Jewels – E-commerce for Jewellery

This repository contains the **M_Jewels** project, an online jewellery shopping website developed using Python and Django.  
The platform is designed to showcase jewellery products, manage customer orders, and provide a smooth online shopping experience.

---

## 📌 Overview

M_Jewels is an e-commerce web application for buying jewellery online. It allows users to browse collections, view product details, add items to cart, place orders, and manage accounts securely.

---

## 🧠 Features

### 👤 User Management
- User registration and login  
- Secure authentication system  
- Manage profile details  
- View order history  

### 💍 Product Management
- Display jewellery collections  
- Categories like rings, necklaces, earrings, bangles, etc.  
- Product images, price, and descriptions  
- Search and filter products  

### 🛒 Shopping Cart
- Add items to cart  
- Update quantity  
- Remove products from cart  
- View total price  

### 💳 Order Management
- Place orders  
- Order confirmation  
- Shipping details  
- Track order status  

### 🛠️ Admin Dashboard
- Add / update / delete products  
- Manage categories  
- View customer orders  
- Update delivery status  
- Manage users  

### 📊 Customer Experience
- Responsive design  
- Easy navigation  
- Attractive product display  
- Smooth checkout process
